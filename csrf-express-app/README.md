# CSRF Express App

This project demonstrates:
- Express session-based authentication
- EJS views for register/login/dashboard
- A CSRF attack simulation page
- CSRF protection using `csurf`
- A minimal JWT example for Task 2

## Run

```bash
npm install
npm start
```

Open `http://localhost:3000`

## JWT example

```bash
node jwtApp.js
```

Open `http://localhost:4000`
